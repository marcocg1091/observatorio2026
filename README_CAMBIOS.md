# Cambios incorporados

Se actualizó el paquete del Observatorio para mejorar la experiencia del mapa interactivo.

## Mejoras principales

- El mapa del Perú ahora tiene regiones SVG con `data-tooltip-chart`.
- Cada región muestra tooltip narrativo con región, valor, indicador, periodo, fuente y explicación ciudadana.
- Los botones de vista del mapa actualizan dinámicamente el indicador: monto convocado, monto adjudicado y monto contratado.
- Se agregó foco de teclado (`tabindex`) para accesibilidad.
- Se agregó panel de ayuda `Mapa interactivo` para orientar al usuario.
- Se añadieron estilos de hover/focus para que el mapa se sienta dinámico e iterativo.

## Nota

Los valores incluidos en el mapa son referenciales para prototipado. En producción deben ser reemplazados por valores provenientes del ecosistema analítico DGDIC o gestionados desde el Admin DGDIC.
